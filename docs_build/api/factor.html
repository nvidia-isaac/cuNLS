Factor API#

The cunls/factor module provides batched residual and Jacobian models for non-linear least squares. Each factor computes a residual vector from one or more state blocks. State blocks lie on manifolds; the factor API uses tangent-space dimensions for Jacobian layout and solver variables. Links to the corresponding state batch types are in the Factor inputs section and in each factor’s Inputs subsection.

Factor inputs#

Each factor’s Inputs subsection below and the State API API list the required state batch types (e.g. VectorStateBatch<Dim>, SO3StateBatch).

Abstract base (cunls/factor/factor_batch.h).
